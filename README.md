# Tic-detection
This is the official implementation of the ICPR 2022 paper [**Facial Tic Detection in Untrimmed Videos of Tourette Syndrome Patients**](https://ieeexplore.ieee.org/abstract/document/9956140).

Only training code is available at this time.

- train_det0102_det.py: code for training TNet (det branch only)
- train_det0102_det_seg.py: code for training TNet
